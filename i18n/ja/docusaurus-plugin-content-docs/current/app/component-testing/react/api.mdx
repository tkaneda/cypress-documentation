## メソッド

### mount

```js
// React 18
import { mount } from 'cypress/react18'

// React 16, 17
import { mount } from 'cypress/react'
```

<table class="api-table table-list">
  <tr>
    <td>説明</td>
    <td>ReactコンポーネントをDOMにマウントする。</td>
  </tr>
  <tr>
    <td>シグネチャ</td>
    <td>
      mount(jsx: React.ReactNode, options?: MountOptions, rerenderKey?: string):
      Cypress.Chainable&lt;MountReturn&gt;
    </td>
  </tr>
  <tr>
    <td>返り値</td>
    <td>Cypress.Chainable&lt;MountReturn&gt;</td>
  </tr>
</table>

<table class="api-table">
  <caption>マウントパラメータ</caption>
  <thead>
    <td>名前</td>
    <td>型</td>
    <td>説明</td>
  </thead>
  <tr>
    <td>jsx</td>
    <td>React.JSX.Element</td>
    <td>マウントするReactコンポーネント</td>
  </tr>
  <tr>
    <td>オプション</td>
    <td>MountOptions(オプション)</td>
    <td>コンポーネントをマウントする際のオプション</td>
  </tr>
  <tr>
    <td>rerenderKey</td>
    <td>文字列型(オプション)</td>
    <td>強制的に再レンダリングするためのキー</td>
  </tr>
</table>

#### 例

```jsx
import { mount } from '@cypress/react'
import { Stepper } from './Stepper'

it('mounts', () => {
  mount(<StepperComponent />)
  cy.get('[data-cy=increment]').click()
  cy.get('[data-cy=counter]').should('have.text', '1')
}
```

### MountOptions

  <caption>メンバ</caption>

  <tr>
    <td>log</td>

    </td>
  </tr>
</table>

### MountReturn

`mount`関数が取得する型
